## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/C-Programming-Concepts.git
2. Navigate to the desired folder and compile the C file:
   ```bash
   gcc filename.c -o output
   ```

3. Run the compiled program:
   ```bash
   ./output
   ```
